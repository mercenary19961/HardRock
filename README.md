## Technology Stack

This repository contains the source code for the HardRock website, built as a modern full-stack SPA.

| Layer | Technology |
|-------|------------|
| Backend | Laravel 12, PHP 8.2+ |
| Frontend | React 18, TypeScript 5 |
| Routing/SPA | Inertia.js 2.0 |
| Styling | Tailwind CSS 3 |
| Animations | CSS Keyframes, Framer Motion |
| Build Tool | Vite 7 |
| Database | MySQL 8.0 |
| Hosting | Railway |
| CDN | Cloudflare |

## Development

```bash
# Setup
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate

# Run locally
php artisan serve     # Laravel server
npm run dev           # Vite dev server
php artisan queue:work # Queue worker

# Production build
npm run build
php artisan optimize
```
